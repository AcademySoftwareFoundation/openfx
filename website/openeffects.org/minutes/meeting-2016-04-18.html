Association General Meeting
UNPUBLISHED

Date: 2016-04-18

Location: Las Vegas Convention Center, Las Vegas NV

Boris Yamnitsky - Boris

Hiew Litt Teen - Blackmagic Design

Jan-Paul Smith - Imagineer Systems

Jim Tierney - Digital Anarchy

Peter Huisma - Assimilate

Peter Litwinowicz - RE:Vision Effects   

Pierre Jasmin - RE:Vision Effects

Rohit Gupta - Blackmagic Design

Stephen Bash - GenArts

Vivek Vedagiri - Blackmagic Design

Minutes

Discussion of holding a technical meeting at IBC with online component. Some were concerned IBC is a busy time and might not be conducive to technical discussions. Agreed anyway to try setting something up and send out invitation emails before IBC.

Discussion of implementing reference host and reference plugins. Concerns were raised regarding validating reference implementations, but some pointed out well targeted (and documented) sample code should be validated by inspection.  Much of the remaining discussion focused on sample plugins, with the recommendation that member organizations should donate examples that demonstrate problems they have run into in the past.  Code samples should also include a discussion of the behavior they are trying to test.  The existing example plugins hosted on GitHub were discussed and the observation made that it can be difficult to contribute to that repository. Black Magic offered to provide example plug-ins for publishing on the website of the Association.

Discussion of creating a certification process. It was pointed out certification has been proposed and dismissed in the past.  Rather than a rigorous certification process, the concept of host support matrices (similar to HTML5 browser support tables) was brought up (see e.g. http://caniuse.com/#feat=canvas ). A plug-in vendor could create a sample plug-in and test that against each host.

It was pointed out that publishing new content on the website / Github of the Association is rather hard. We need to make publishing simpler. No explicit action item.

Vote to continue current directors was unanimous.

Pierre Jasmin took an action to audit Google Group Membership list to make sure meeting announcements reach all member organizations.

Discussion of possibly making membership dues optional. It was pointed out that two open source OFX hosts are already non-paying members.
