Association General Meeting
Published

Date: 2020-04-21

Location: Zoom teleconference

Gary Oberbrunner (Dark Star Systems), Dennis Adams (Sony), Pierre Jasmin (ReVision Fx), Paul Miller (BFX), Rohit Gupta (BMD), Phil Barrett (Filmlight), Paul Miller (BFX), Boris Yamnitsky (BFX), jclement (BFX), Jason Clement (BFX), JP Smith (BFX), Larissa Supnik (BFX), Peter Loveday (BMD), Rohit Gupta (BMD), Thomas True, Vivek Vedagiri (BMD)

Jasmin Roy from AD (Flame OpenGL->Vulkan), Manuel & Pirmin (LIT Labs), Philippe Jean (Flame), Alexandre Bizeau (36pix), Delia Hou (NVIDIA), Dee Fussell (MAGIX), Alexandre Gauthier (Left Angle), Delia (?), JT Nelson (Pasadena OpenSource), Mikki Wells, Pirmin (LIT Labs), 

Minutes

Admin agenda:

1. Peter Huisma presented bank and finance update. 

2. Peter Huisma presented an overview of the Association, how to join, and requested a vote to renew the board membership.

3. Confirmation of organization structure. Directors are Peter Huisma, Pierre Jasmin, Gary Oberbrunner, Dennis Adams, and JP Smith. Proposed that this continues, seconded. Vote passes unanimously.

Closed admin discussion.

 

API discussion:

Gary Oberbrunner gave a presentation on how to contribute to the OFX Specification using github. A new process for discussing and approving standard changes using github issues was discussed and approved.

Pierre Jasmin led a discussion of the open API enhancements.

Dennis Adams led a discussion of GPU enhancements:
- Overlays and interacts
- GPU utilities
- OpenCL Render extension
- Canonicalization of GPU render entry points
 

Meeting adjourned.
