Progress Suite - Documentation to be added Proposed

Standard version: 1.4

Major Change

We amend the documentation of progress suite so that it is expected that it can be raised in a modal manner and have a ‘cancel’ button. Plugins that perform analysis post an appropriate message, raise the progress monitor in a modal manner and should poll to see if processing has been aborted. Any cancellation should be handled gracefully by the plugin (eg: reset analysis parameters to ‘default’ values).

Many hosts already do that (e.g. Vegas, Nuke, Natron,...).

 

This may have some threading issues on host application (?)
