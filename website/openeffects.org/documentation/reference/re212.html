kOfxParamPropParametricDimension
Prev   Chapter 17. Properties By Name   Next

Name

kOfxParamPropParametricDimension — The dimension of a parametric param

Synopsis

#include "/ofxParametricParam.h"
#define kOfxParamPropParametricDimension "OfxParamPropParametricDimension"

Type

kOfxParamPropParametricDimension is of type int , and has only a single dimension.

Property Set

This is a property that belongs to the parametric param descriptor (read/write) and instance (read only).

Description

The dimension of a parametric param

  • default - 1
  • Value Values - greater than 0

This indicates the dimension of the parametric param.
